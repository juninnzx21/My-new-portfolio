Why this project was created

This project was built as a modern frontend application using Vue 3, TypeScript, and Vite to demonstrate a cleaner component-based approach for displaying and organizing product data.

It consumes an API endpoint, supports product search, and provides sortable product columns in a responsive interface.

It highlights my ability to work with modern frontend tooling, reusable components, API consumption, and production builds for static deployment.
