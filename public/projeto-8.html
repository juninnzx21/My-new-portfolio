Why this project was created

Market Decision AI was created as a decision-support platform for trading analysis, focused on realistic risk, signal validation, operational timing, and live market intelligence.

The stack combines a Vue frontend with a FastAPI backend, structured engines for technical and historical scoring, WebSocket live board updates, and a modular analysis pipeline.

It demonstrates product thinking, data presentation, API integration, real-time dashboards, and a stronger system design than a simple static trading page.
